# Halal Box Jo - Angular Frontend

A modern one-page Angular frontend for **Halal Box Jo**, inspired by the provided menu and branding images.

## Included
- Hero section with Halal Box Jo logo
- Combo menu section using the same food photos from the provided images
- Family meals section
- Shrimp add-ons section
- Contact section with phone and location
- Responsive layout

## Run locally
```bash
npm install
npm start
```

Then open the local Angular dev server in your browser.

## Notes
The food images were extracted from the provided menu artwork and added under:
`src/assets/images`
